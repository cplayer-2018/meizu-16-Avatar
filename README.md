## Html 操作步骤
1. 使用 FireFox 浏览器访问此 Html 文件。
2. 依次点击“web 开发者”-“响应式设计模式”，并将尺寸设置为 520X520。
3. 点击“截取视区(viewport)图像”(相机图标)，后点击保存路径并保存。
